Bonjour, je suis Benjamin Duponchel, en formation TSSR. Passionné par les systèmes et réseaux, voici quelques projets réalisés.

Enoncé du projet

Votre entreprise est en pleine croissance et doit emménager dans de nouveaux locaux plus grands. Vous profitez de l'installation dans ces nouveaux locaux pour revoir la configuration du réseau et améliorer la sécurité.

L’entreprise est constituée de cinq départements :
▪ Le service Relations Humaines (qui inclut la direction) (5 personnes)
▪ Le service comptabilité (3 personnes)
▪ Le service commercial (13 personnes)
▪ Les équipes opérationnelles (52 personnes)
▪ L’équipe informatique (2 personnes) <- c’est votre équipe !

L’entreprise a aussi 3 départements sur un site distant :
▪ Le service R&D (9 personnes)
▪ Le service des achats (27 personnes)
▪ Le service SAV (7 personnes)

Le réseau de l’entreprise est sur 192.168.100.0/24
Le réseau distant est sur le 172.16.0.0/24
Le réseau intersites est sur le 10.0.0.0/8

A faire :
• Définir un plan d’adressage IP pour chaque site
• Faire un schéma sur draw.io
Et sur Cisco PacketTracer :
• Configurer les VLAN sur chaque site
• Configurer le routage InterVLAN sur chaque site
• Faire communiquer les deux sites via des « ping »
