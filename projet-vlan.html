Projet de Configuration Réseau

Contexte

Votre entreprise est en pleine croissance et doit emménager dans de nouveaux locaux plus grands. Vous profitez de l'installation dans ces nouveaux locaux pour revoir la configuration du réseau et améliorer la sécurité.

Départements

  • Service Relations Humaines (inclut la direction) : 5 personnes
  • Service comptabilité : 5 personnes
  • Service commercial : 20 personnes
  • Équipes opérationnelles : 49 personnes
  • Équipe informatique : 3 personnes

Site distant

  • Service R&D : 11 personnes
  • Service des achats : 31 personnes
  • Service SAV : 6 personnes

Plan d'adressage IP

  • Réseau principal : 192.168.100.0/24
  • Réseau distant : 172.16.0.0/24
  • Réseau intersites : 10.0.0.0/8

Tâches à réaliser

  1. Définir un plan d’adressage IP pour chaque site
  2. Faire un schéma réseau sur Draw.io
  3. Configuration sur Cisco PacketTracer
    • Configurer les VLAN sur chaque site
    • Configurer le routage InterVLAN sur chaque site
    • Faire communiquer les deux sites via des ping
